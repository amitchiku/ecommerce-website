
MERN eCommerce Website

A modern full-stack eCommerce web application built using the MERN stack with authentication, product management, shopping cart, order system, and admin dashboard functionality.

User Authentication & Authorization

Shopping Cart Functionality
Secure Checkout Flow
Admin Dashboard
Product Management
Order Management
Responsive UI Design
REST API Integration
MongoDB Database Integration

React.js
Redux Toolkit
Axios
React Router
